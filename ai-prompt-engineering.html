AI คิดอย่างไร?

AI ไม่ได้ "รู้" สิ่งที่คุณต้องการโดยอัตโนมัติ มันจะ ตีความ จากคำที่คุณพิมพ์เท่านั้น ยิ่ง Prompt ชัดเจน ยิ่งได้ผลลัพธ์ตรงใจ

❌ คลุมเครือ
"ช่วยเรื่องงาน"
✅ ชัดเจน
"เขียนอีเมลขอโทษลูกค้า ภาษาสุภาพ 3 ย่อหน้า เรื่องส่งของล่าช้า 2 วัน"
📐

สูตร RCTF — โครงสร้าง Prompt ที่ดี

Role — บทบาทของ AI
Context — บริบท / ข้อมูลพื้นฐาน
Task — งานที่ต้องทำ (ชัดเจน)
Format — รูปแบบผลลัพธ์ที่ต้องการ

ตัวอย่าง: "คุณคือครูประถมศึกษา (Role) นักเรียนอายุ 8 ปี เรียนภาษาไทย (Context) สร้างแบบฝึกหัดเติมคำในช่องว่าง (Task) จำนวน 5 ข้อ ในรูปแบบ HTML ที่พิมพ์ออกได้ (Format)"

💡

5 หลักการที่ต้องจำ

  • เฉพาะเจาะจง (Specific)

    ระบุรายละเอียดให้ครบ: ใคร อะไร กี่อัน ยาวแค่ไหน ใช้ทำอะไร

  • ให้บริบท (Context)

    บอก AI ว่าใครเป็นผู้รับผลลัพธ์ สถานการณ์คืออะไร

  • ระบุรูปแบบ (Format)

    ต้องการ bullet list, ตาราง, โค้ด, หรือย่อหน้า? บอกให้ชัด

  • ให้ตัวอย่าง (Examples)

    ถ้ามีตัวอย่างงานที่ชอบ ใส่เข้าไปใน Prompt เลย

  • ทดสอบ & ปรับ (Iterate)

    ไม่ได้ผลครั้งแรก? ปรับ Prompt และลองใหม่ — เป็นเรื่องปกติมาก

Chain-of-Thought — ให้ AI คิดทีละขั้น

เพิ่มวลี "คิดทีละขั้นตอน" หรือ "อธิบายเหตุผลก่อนตอบ" ช่วยให้ AI แก้ปัญหาซับซ้อนได้ดีขึ้น

❌ ธรรมดา
"ข้อสอบนี้ตอบว่าอะไร?"
✅ Chain-of-Thought
"วิเคราะห์โจทย์นี้ทีละขั้น แล้วระบุคำตอบพร้อมเหตุผล"
🎭

Role Prompting — กำหนดบทบาท

บอก AI ว่าควรเป็น "ใคร" เพื่อปรับน้ำเสียง ความเชี่ยวชาญ และสไตล์การตอบ

🔹 "คุณคือนักออกแบบ UX มืออาชีพ" → ตอบในมุมมองการใช้งาน
🔹 "คุณคือครูสอนภาษาไทย ป.3" → ใช้ภาษาเด็กเข้าใจง่าย
🔹 "คุณคือนักวิเคราะห์ข้อมูล" → ให้ตัวเลข สถิติ เหตุผล
📝

Few-Shot Prompting — สอนด้วยตัวอย่าง

ใส่ตัวอย่าง input → output ก่อน แล้วให้ทำรูปแบบเดิม

ตัวอย่าง:
คำ: แมว → ประโยค: "แมวตัวนี้ขนนุ่มมาก"
คำ: ดอกไม้ → ประโยค: "ดอกไม้ในสวนบานสะพรั่ง"

ตอนนี้ทำเหมือนกัน:
คำ: ท้องฟ้า → ประโยค: ???
🚫

Negative Prompting — บอกสิ่งที่ไม่ต้องการ

ระบุสิ่งที่ ไม่ต้องการ ช่วยตัดสิ่งที่ไม่เกี่ยวออก

❌ ไม่มี Constraint
"อธิบาย AI ให้ฉันฟัง"
✅ มี Constraint
"อธิบาย AI ให้นักเรียน ป.4 ฟัง ไม่ใช้ศัพท์เทคนิค ไม่เกิน 5 ประโยค"
🔄

Prompt Chaining — แบ่งงานใหญ่เป็นชิ้นเล็ก

  • Prompt 1: วางโครงร่าง

    "วางหัวข้อหลัก 5 ข้อสำหรับบทเรียนเรื่อง..."

  • Prompt 2: ขยายแต่ละหัวข้อ

    "นำหัวข้อที่ 2 มาเขียนเป็นเนื้อหา 2 ย่อหน้า"

  • Prompt 3: สร้างแบบฝึกหัด

    "จากเนื้อหาด้านบน สร้างคำถาม 3 ข้อ"

สร้างแผนการสอน

Prompt:
"คุณคือครูภาษาไทยประถมศึกษาปีที่ 3 สร้างแผนการสอนเรื่อง 'การแต่งประโยค' สำหรับนักเรียน 30 คน เวลา 50 นาที

รูปแบบ:
- จุดประสงค์การเรียนรู้ (3 ข้อ)
- กิจกรรมแบ่งเป็น 3 ช่วง (นำเข้า / สอน / สรุป)
- สื่อที่ใช้
- การวัดผล

ใช้กิจกรรมที่สนุก เน้นการมีส่วนร่วม"
✓ มี Role ✓ มี Context ✓ ระบุ Format ✓ ระบุ Tone
✍️

สร้างข้อสอบ

Prompt:
"สร้างข้อสอบปรนัย 4 ตัวเลือก จำนวน 10 ข้อ วิชาวิทยาศาสตร์ ป.4 เรื่อง 'ระบบสุริยะ'

กระจายระดับ: ความจำ 3 ข้อ, ความเข้าใจ 4 ข้อ, ประยุกต์ 3 ข้อ
ออกเป็น JSON: { id, question, choices[4], answer, level }
ใช้ภาษาที่เด็ก ป.4 เข้าใจได้"
🎮

สร้างเกมการศึกษา (สำหรับ Dev)

Prompt:
"สร้าง HTML เกม Quiz แบบ Single-file ไม่ใช้ Library ภายนอก

Spec:
- คำถามสะกดคำภาษาไทย 5 ข้อ
- แสดงตัวเลือก 4 ตัว สุ่มตำแหน่ง
- นับคะแนน + แสดง Progress bar
- เสียง feedback ด้วย Web Audio API (ถูก=เสียงสูง, ผิด=เสียงต่ำ)
- Responsive ใช้ได้บนมือถือ
- สีหลัก: น้ำเงิน #1e40af"
📧

เขียนเอกสารราชการ

Prompt:
"ร่างหนังสือราชการภายนอก จาก: ผู้อำนวยการโรงเรียนบ้านคำไผ่ ถึง: สพป.อุดรธานี เขต 1

เรื่อง: ขอรับการสนับสนุนงบประมาณซ่อมแซมห้องสมุด
รายละเอียด: ห้องสมุดชำรุด หลังคารั่ว ต้องการงบ 150,000 บาท

ใช้รูปแบบหนังสือราชการไทยมาตรฐาน ภาษาทางการ"
