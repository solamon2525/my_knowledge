Claude Code CLI

คู่มือการติดตั้งและการใช้งานผู้ช่วยเขียนโค้ดอัจฉริยะบน Terminal จาก Anthropic

1. Claude Code CLI คืออะไร?

Claude Code คือ Agentic Command-Line Interface (CLI) จาก Anthropic ที่ออกแบบมาเพื่อช่วยนักพัฒนาในการค้นหาข้อมูล แก้ไขโค้ด ทดสอบระบบ และจัดการ Codebase ได้โดยตรงจาก Terminal ของคุณ

ฟีเจอร์หลัก (Key Features)

  • Agentic Editing: แก้ไขโค้ดหลายไฟล์พร้อมกัน, Refactor และแก้บั๊กได้อัตโนมัติ (รอการอนุมัติจากคุณ)
  • Toolchain Access: รันคำสั่ง Shell, รัน Unit Test และอ่าน/เขียนไฟล์ในเครื่องได้
  • Subagents: มอบหมายงานเฉพาะทาง (เช่น งานวิจัยหรือวางแผน) ให้โมเดลย่อยทำงานคู่ขนานได้ เพื่อประหยัด Context หลัก
  • MCP Integration: รองรับ Model Context Protocol ในการเชื่อมต่อเครื่องมือ ฐานข้อมูล หรือ API ภายนอก
  • Git Integration: สร้าง Checkpoints อัตโนมัติ, จัดการ Branch และช่วยร่าง Commit message

2. การติดตั้ง (Installation)

ระบบต้องการ Node.js 18+ และสามารถใช้งานได้ทั้งบน macOS, Linux และ Windows (ผ่าน WSL หรือ PowerShell)

สำหรับ Windows (PowerShell)

irm https://claude.ai/install.ps1 | iex

สำหรับ macOS / Linux / WSL

curl -fsSL https://claude.ai/install.sh | bash

ทางเลือก: ติดตั้งผ่าน npm

npm install -g @anthropic-ai/claude-code

3. การเริ่มต้นเปิดใช้งาน (Getting Started)

  1. เข้าสู่โฟลเดอร์โปรเจกต์: 
    cd path/to/your-project
  2. เริ่มการทำงาน: 
    claude
    *ในการใช้งานครั้งแรก ระบบจะเปิดเบราว์เซอร์เพื่อให้คุณ Login ด้วยบัญชี Anthropic

4. คำสั่งและการสั่งงาน (Commands & Usage)

คำสั่งระบบ (Slash Commands)

พิมพ์ / เพื่อเรียกใช้งานคำสั่งระบบที่สำคัญ:

  • /help : แสดงรายการคำสั่งทั้งหมดที่มี
  • /compact : (สำคัญ) รวบตึงประวัติการสนทนาเพื่อลดการใช้ Token และประหยัดค่าใช้จ่าย
  • /clear : ล้างหน้าจอ Terminal
  • /buddy : เรียกสัตว์เลี้ยง ASCII มาอยู่เป็นเพื่อน (Easter egg)
  • /logout : ออกจากระบบและลบ Credential
  • exit (หรือกด Ctrl+C) : ออกจากโปรแกรม

การสั่งงานด้วยภาษาธรรมชาติ

คุณสามารถสั่งงานเป็นประโยคได้เลย เช่น:

  • Explain what this file does
  • Find and fix the bug in [function name]
  • Write unit tests for the utility module
  • Run the build and tell me if it fails

5. ความปลอดภัยและการตั้งค่า (Security & Config)

  • ระบบ Permission: ทุกครั้งที่ Claude จะแก้ไขไฟล์หรือรันคำสั่ง Shell ระบบจะหยุดและถามคุณก่อน [y/n] เสมอ
  • Undo & Checkpoints: มีระบบ Checkpoint ผ่าน Git อัตโนมัติ หากโค้ดพังสามารถสั่ง "Undo" กลับไปสถานะเดิมได้ทันที
  • CLAUDE.md: แนะนำให้สร้างไฟล์ CLAUDE.md ไว้ที่ Root ของโปรเจกต์ เพื่อใส่กฎเกณฑ์ (Coding Standards) และบริบทเฉพาะของโปรเจกต์ให้ AI ทำงานได้ตรงใจที่สุด
