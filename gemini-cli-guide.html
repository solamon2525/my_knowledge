Gemini CLI - พลัง AI ใน Terminal

คู่มือการใช้งานผู้ช่วยวิศวกรซอฟต์แวร์อัจฉริยะที่รันตรงบนเครื่องคุณ

1. Gemini CLI คืออะไร?

Gemini CLI คือเครื่องมือบรรทัดคำสั่ง (Command Line Interface) ที่พัฒนาโดย Google เพื่อให้นักพัฒนาสามารถใช้งานโมเดล Gemini 1.5 Pro/Flash ได้โดยตรงจาก Terminal ของเครื่องตนเอง มีความสามารถเหนือกว่าแชทบอททั่วไปเพราะสามารถ เข้าถึงไฟล์ในเครื่อง รันคำสั่ง Shell และค้นหาข้อมูลจากอินเทอร์เน็ตได้

2. การติดตั้ง (Installation)

คุณต้องมี Node.js ติดตั้งอยู่ในเครื่องก่อน จากนั้นรันคำสั่ง:

npm install -g @google/gemini-cli

หรือถ้าต้องการรันชั่วคราวโดยไม่ติดตั้ง:

npx @google/gemini-cli

3. การเริ่มต้นใช้งาน

เพียงพิมพ์คำสั่งสั้นๆ ใน PowerShell หรือ Terminal:

gemini

ในครั้งแรก โปรแกรมจะขอให้คุณใส่ API Key ซึ่งคุณสามารถรับได้ฟรีจาก Google AI Studio

4. ฟีเจอร์เด่น (Key Features)

  • วิเคราะห์โค้ด (Code Analysis): สั่งให้อ่านไฟล์และสรุปการทำงานหรือหาจุดบกพร่องได้ทันที
  • รันคำสั่ง Shell: สั่งให้ AI รันคำสั่งเช่น git status, npm test หรือจัดการไฟล์ได้โดยตรง
  • Google Search Integration: AI สามารถค้นหาข้อมูลล่าสุดจากอินเทอร์เน็ตเพื่อตอบคำถามที่แม่นยำขึ้น
  • Multimodal: รองรับการประมวลผลทั้งข้อความ โค้ด และรูปภาพ (ถ้ามีไฟล์ภาพในโปรเจกต์)

6. คำสั่งลัด (Slash Commands)

คำสั่ง หน้าที่
/help แสดงรายการคำสั่งทั้งหมด
/bug รายงานปัญหาการใช้งาน
exit ออกจากโปรแกรม
