Build with Claude

ศูนย์รวมส่วนขยาย ทักษะ และเครื่องมือสำหรับนิเวศน์วิทยาของ Claude AI

1. Build with Claude คืออะไร?

เป็น Marketplace และ Registry สำหรับเครื่องมือที่ช่วยเพิ่มความสามารถให้กับ Claude AI (เช่น Claude.ai, Claude Code และ API) โดยรวบรวมส่วนขยายที่พร้อมใช้งานมากกว่า 500 รายการ

2. หมวดหมู่เครื่องมือหลัก

🛠️ Skills (130+)

ทักษะการเชื่อมต่อกับแอปภายนอก เช่น Google Drive, GitHub, Slack

🤖 Subagents (110+)

AI Persona เฉพาะทาง เช่น ผู้เชี่ยวชาญด้าน Security หรือ Data Analyst

⌨️ Commands (170+)

ชุดคำสั่งสำเร็จรูปสำหรับ Claude Code CLI เพื่อเร่งความเร็วการพัฒนา

🔗 Plugins & Hooks

ส่วนขยายสำหรับจัดการ Workflow อัตโนมัติในระหว่างการเขียนโค้ด

3. มาตรฐานที่ใช้ (MCP)

เครื่องมือส่วนใหญ่พัฒนาบนมาตรฐาน Model Context Protocol (MCP) ซึ่งช่วยให้ AI สามารถเข้าถึงข้อมูลและเครื่องมือจากแหล่งต่างๆ ได้อย่างปลอดภัยและเป็นมาตรฐานเดียวกัน

4. ประโยชน์สำหรับนักพัฒนา

  • ประหยัดเวลาในการเขียนโค้ดเชื่อมต่อ (Integrations) เอง
  • เพิ่มความสามารถเฉพาะด้านให้กับ AI เพื่อทำงานที่ซับซ้อนขึ้น
  • ใช้คำสั่ง CLI สำเร็จรูปเพื่อจัดการงานที่ทำซ้ำๆ เช่น การสร้างระบบ Auth

เข้าชมเว็บไซต์อย่างเป็นทางการเพื่อค้นหาเครื่องมือ
